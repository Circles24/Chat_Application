# Chat_Application
### Simple chat application :smile:

## Installation

In case you don't have git

 `sudo apt-get install git`
 
Clone the repo
 
 `git clone https://github.com/Circles24/Chat_Application.git`
 
Open the directory

`cd Chat_Application`

Give executable permission

`chmod +x *.sh`

## Usage

### Building the application

`./Build.sh`

### Run

#### Start the server

`./Server.sh`

#### Start a client

`./Client.sh`
